git 分为3个区

* 工作区 —— 存放文件
* 暂存区 —— 工作区变更，提交到暂存区
* 版本库 —— 暂存区的内容，提交到版本库

status颜色
* 红: 工作区变更了,但是**没有提交到暂存区**,此时 `git checkout .` 会撤回变更无法找回
* 绿: 已经提交到暂存区
* 无: 都被git管理起来了

```bash
git status # 查看文件状态（红）
 
git add . # 工作区-> 暂存区（变绿）

git rest HEAD # 把暂存区 -> 工作区（由红红变绿）

git checkout . #工作区 变更撤回
```

```bash
git commit -m '注释' # 暂存区 -> 版本库

```
